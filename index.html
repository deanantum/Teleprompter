Control Mode

Second monitor (mirror window)

If you work on the 2nd monitor and want the mirror on the main one, choose Left. Otherwise choose Right, Above, or Below.

If the mirror window doesn’t move to the other monitor by itself, drag it there. On Mac and in some setups the browser blocks automatic placement.

For automatic placement when supported, allow multi-screen access when your browser asks. If the button below says "Not available", use Left/Right above and drag the mirror if needed. From file (file://), run npx serve then open http://localhost:3000.
